Разбиение текста на предложения

Machine Learning, алгоритмы, аналитика

Суть: разбить сплошной текст на предложения
План действий:
1.	Простым способом найти в тексте места-кандидаты на границу двух предложений, например «точка»( в файле функция sentence_tokenizer)
2.	Вручную разметить некоторую обучающую выборку примеров реальных границ предложений(Сделала по примеру английского корпуса на https://github.com/nlp-compromise/penn-treebank в файле CreatTrainTest)
3.	Описать кандидаты каким-то числовым вектором признаков(Сделано в файле SenTok функцией trigram_featurizer)
4.	Провести классификацию на два класса:(Обучила модель на базе алгоритма SVM в файле TrainSBModel)
    a.	Действительно являются границами предложений
    b.	Не являются границами предложений
